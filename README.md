# zombie-release

## A theme for Jellyfin 10.9.x combining the best from the Public Themes

Huge Thanks to all the theme creators out there for their hard work. This theme takes bits and pieces from other themes, namely [Zesty](https://github.com/stpnwf/ZestyTheme/tree/main), [JellySkin](https://github.com/prayag17/JellySkin/tree/master), [Ultrachromic](https://github.com/CTalvio/Ultrachromic) and [JellyFin Landscape](https://github.com/krlsantcard/Jellyfin-10.9/tree/main)
